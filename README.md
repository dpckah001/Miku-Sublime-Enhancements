# 🎵 Miku-Sublime-Enhancements 🎵
> 将你的 Sublime Text 3 变成初音未来的专属工作站。

这是一个为 **Sublime Text** 深度定制的增强套件，包含启动特效、初音色调主题以及“隐形”摸鱼阅读器。

## ✨ 功能特性

* **⚡ Miku Boot System**: 启动时自动触发系统自检序列，带进度条和 ASCII 艺术。
* **📖 Miku Reader (Stealth Mode)**: 
    * `Alt+Space`: 在代码弹窗中阅读小说（模拟 Docstring，极其隐蔽）。
    * `Ctrl+Alt+N`: 抓取知乎/科技热点，伪装成项目 Readme。
* **⌨️ HUD Keymap**: `F10` 调起 Miku 配色的快捷键速查面板，带淡出动画。
* **🎨 Miku Theme**: 专属葱青色 (#39C5BB) 高亮方案。

## 🚀 安装方法

1.  打开 Sublime Text，点击 `Preferences -> Browse Packages...`。
2.  克隆或下载本仓库内容。
3.  将文件移动到 `User` 目录下。
4.  在 `Key Bindings` 中添加仓库中推荐的快捷键配置。

## 🎹 快捷键配置 (Recommended)
| 快捷键 | 功能 |
| :--- | :--- |
| `Alt + Space` | 阅读下一行小说 |
| `Ctrl + Alt + N` | 调起科技快报 |
| `F10` | 呼唤速查面板 |
